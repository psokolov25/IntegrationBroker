# Agents.md

Дублирующий файл для совместимости с разными агентами/инструментами.

Содержимое должно оставаться синхронизированным с `AGENTS.md`.

Ключевые принципы:

- Контракт `InboundEnvelope` неизменяемый и расширяемый (добавление полей — назад-совместимо).
- Все новые интеграции — через типизированные Java-адаптеры и интерфейсы.
- Любые изменения, влияющие на идемпотентность/outbox/DLQ, требуют отдельного тестового покрытия.
